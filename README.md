# COVID-19 Comparative Analysis: Italy vs Pakistan

**Author:** Abu Bakar

---

## 📌 Project Overview
This project analyzes and compares COVID-19 trends between Italy and Pakistan using real-world data from *Our World in Data*.

The goal is to understand differences in:
- Total COVID-19 cases over time  
- Cases per million population  
- Death rates  
- Vaccination progress  
- Epidemic waves  

---

## 📊 Dataset
Source: https://ourworldindata.org/coronavirus-data

The dataset includes global COVID-19 statistics such as:
- Confirmed cases  
- Deaths  
- Vaccinations  
- Population data  

---

## 🧰 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  

---

## 📈 Key Insights

- Italy experienced an earlier and faster COVID-19 outbreak compared to Pakistan  
- Cases per million were significantly higher in Italy  
- Vaccination rollout was faster in Italy  
- Both countries show multiple infection waves over time  

---

## 📷 Methodology
1. Data collection from OWID dataset  
2. Filtering Italy and Pakistan  
3. Data cleaning and handling missing values  
4. Feature engineering (cases per million, death rate, etc.)  
5. Visualization using Matplotlib  
6. Interpretation of results
